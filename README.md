# Travel Insurance Automation Test

## Description
Automation test written in Java using Selenium WebDriver and TestNG.
The test covers a basic purchase flow of travel insurance.

## Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven

## Test Scenario
1. Open travel insurance page
2. Start purchase process
3. Select continent
4. Select travel dates using date picker
5. Validate total days
6. Proceed to passenger details

## How to Run
1. Clone repository
2. Open project in IntelliJ
3. Make sure ChromeDriver exists in /drivers
4. Run test with TestNG
